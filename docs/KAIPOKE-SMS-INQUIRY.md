# SMS社（カイポケ）への照会ドラフト（ロードマップ G2）

> 目的: ブラウザ拡張による「転記支援（半自動入力）」がカイポケの利用規約上問題ないかを、
> 本番運用の前に確認する（docs/P2-KAIPOKE.md: 明示的な禁止条項は確認できなかったが、
> 一般条項があるため照会推奨）。
> 送付タイミング: 実機検証で技術的に動くことを確認した後、**実証運用の開始前**（吉本さん判断）。
> 送り先: カイポケのサポート窓口（会員向け問い合わせ）。

## 送信文面（そのまま使えます・適宜調整OK）

件名: ブラウザ拡張機能による入力補助ツールの利用可否について

カイポケ サポートご担当者様

いつもお世話になっております。カイポケを利用している［事業所名／会員ID］の吉本と申します。

居宅介護支援の記録業務の負担軽減のため、自社利用を目的とした入力補助ツール
（Chrome/Edge のブラウザ拡張機能）の利用を検討しており、貴社サービスの利用規約上
問題がないかを確認させてください。

【ツールの動作内容】
1. 職員が別画面で作成した記録の下書きを、カイポケの帳票編集画面の該当のテキスト欄に
   入力します（職員がボタンを押した時のみ動作する、コピー＆ペーストの代替です）。
2. 入力後の内容確認と「登録（保存）」ボタンの操作は、必ず職員が手動で行います
   （自動での保存・確定は行いません）。
3. カイポケの画面情報・データを外部に送信することはありません。
4. ログインは通常どおり職員が行い、ID・パスワードをツールが保存・使用することはありません。
5. サーバーへの機械的な連続アクセス（スクレイピング・クローリング）は行いません。

【確認したい点】
- 上記のような、職員の操作を補助する範囲でのブラウザ拡張の利用は、
  利用規約上問題ないでしょうか。
- 問題がある場合、どのような条件であれば許容されるか（または代替手段のご案内が
  あれば）ご教示ください。

お手数をおかけしますが、よろしくお願いいたします。

［氏名・事業所名・会員ID・連絡先］
