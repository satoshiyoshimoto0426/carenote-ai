# CareNote AI ブラウザ拡張（カイポケ連携）

面談メモから帳票の下書きをAIで作り、**サイドパネル**に表示して、
カイポケの編集画面へ**ワンクリックで流し込む**ための Chrome/Edge 拡張（Manifest V3）です。

- Step1（転記支援）: 下書きをセクション単位で表示し、ワンクリックでコピー。どのソフトでも使えます。
- Step2（流し込み）: カイポケの該当テキスト欄へ自動入力。**保存・確定は必ずご自身で**行ってください。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | 拡張の定義（権限・対象サイト `r.kaipoke.biz`） |
| `src/panel.html` / `panel.css` / `panel.js` | サイドパネル本体（生成・表示・コピー・流し込み起点） |
| `src/content.js` | カイポケ画面に常駐し、パネルからの依頼で欄へ書き込む |
| `src/adapters/kaipoke.js` | CareNote下書き → カイポケ欄のマッピングと書込ロジック |
| `src/background.js` | アイコンクリックでサイドパネルを開く |
| `src/options.html` / `options.js` | API接続先・トークンの設定 |

## 入れ方（開発・検証用：パッケージ化しない読み込み）

1. Chrome（または Edge）で `chrome://extensions` を開く。
2. 右上の「**デベロッパー モード**」をオンにする。
3. 「**パッケージ化されていない拡張機能を読み込む**」を押し、この `extension` フォルダを選ぶ。
4. ツールバーに拡張アイコンが出ます（固定しておくと便利）。

## 最初の設定

1. 拡張アイコンを右クリック →「オプション」、またはパネル右上の「設定」を開く。
2. **API のベースURL**: CareNote AI を公開しているURL（例: Vercel の本番URL）。
3. **拡張トークン**: サーバ側の環境変数 `CARENOTE_EXTENSION_TOKEN` と同じ値。
4. 「保存」→「接続テスト」で「接続できました」と出れば準備完了です。

> トークンとURLはこの端末のローカルにのみ保存され、生成API以外には送信しません。

## 使い方

1. カイポケにログインし、入力したい帳票の**編集画面を開く**。
2. 拡張アイコンを押してサイドパネルを開く。
3. 帳票の種類を選び、面談メモなどを入れて「**下書きを生成**」。
   - すでにWeb版「作成」ページで作った下書きがあれば、JSONを貼り付けて読み込むこともできます。
4. 各セクションの「コピー」でカイポケへ貼り付け（Step1）。
5. アセスメント／ケアプラン第1表では「**この画面に流し込む**」で該当欄へ自動入力（Step2）。
   - ハイライトされた内容を確認し、**カイポケの保存ボタンはご自身で**押してください。

## 対応している自動入力の範囲（2026-06-15 時点）

| 帳票 | 自動入力できる欄 |
|---|---|
| アセスメント | 主訴・意向（本人欄）※家族分は要分割 / 生活歴・経過 / 全体のまとめ |
| ケアプラン第1表 | 意向を踏まえた課題分析の結果 / 総合的な援助の方針 |

チェック・ラジオの項目やケアプラン第2表は、画面構造（DOM）を追加取得してから対応予定です。
それまではコピー（Step1）でご利用ください。

## 注意（安全のために）

- AIの出力は**下書き**です。確定・保存の前に必ずケアマネジャーが確認してください。
- 拡張はカイポケのログイン情報を保存・送信しません。画面の内容も外部に送りません。
- カイポケの画面構成が変わると流し込みが効かなくなることがあります（その場合はコピーをご利用ください）。
