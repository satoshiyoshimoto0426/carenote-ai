## 書き方（1ファイル＝1テーマ、1ルール＝4項目）

`_TEMPLATE.md` をコピーして書く。1つのルールは必ずこの4項目：

- **状況**: いつ・どの帳票のどの欄の話か
- **禁止**: やってはいけない書き方（NG例があれば1つ）
- **推奨**: こう書く（OK例があれば1つ。文例はオリジナルで）
- **根拠**: なぜか（経験則なら「実地指導で指摘されやすい」等。公式文書なら出典名）

## フォルダ構成

```
knowledge/
├─ 第1表/    ├─ 第2表/    ├─ 第3表/（週間計画・任意）
├─ 第4表/    ├─ 第5表/    ├─ 第6-7表/（給付管理。生成対象外だが知見は貯めてよい）
├─ アセスメント/  ├─ モニタリング/  └─ 共通/（NG表現集・文体・様式横断のルール）
```

## 公式ルールブックの取り込み方（著作権・重要）

- **法令・告示・通達**（例: 老企第22号 等）→ 著作権法13条で保護対象外。**そのまま引用OK**（出典名を書く）
- **手引き・マニュアル・白書**（厚労省・自治体の解説物）→ 著作権あり。**全文コピペ禁止**。
  自分の言葉で**要旨をルール化**し、「出典: ◯◯手引き p.X（要旨）」と書く
- 迷ったら要旨化しておけば安全
